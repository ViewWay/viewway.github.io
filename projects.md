## UCEF — Universal Context Extension Framework

**What:** A model-agnostic framework that enables any LLM to process documents exceeding its context window.

**How:** Hyperbolic retrieval + quantum-inspired selection + adaptive compression + quality feedback loop.

**Stack:** Python, NumPy, SciPy, Zhipu API, DeepSeek API, LongBench

**Links:**
- [GitHub](https://github.com/ViewWay/UCEF)
- [Documentation Site](https://viewway.github.io/UCEF/)

## FeMeter — Three-Phase Smart Energy Meter Firmware

**What:** Rust firmware for FM33A0xx (Cortex-M0+) three-phase smart electricity meters.

**Features:** 6×UART + 2×LPUART, FreeRTOS 11.x, DL/T645 protocol, ~10,400 lines

**Stack:** Rust, embedded, thumbv6m-none-eabi

**Links:**
- [GitHub](https://github.com/ViewWay/FeMeter)
